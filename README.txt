Collection PWA

GitHub Pages 上傳方式：
1. 解壓縮 ZIP
2. 將所有檔案上傳到 GitHub Repository
3. Repository → Settings → Pages
4. Branch 選 main / root
5. 開啟 GitHub Pages 網址

iPhone 安裝：
1. 使用 Safari 開啟 GitHub Pages 網址
2. 點分享
3. 加入主畫面
4. 從主畫面開啟 Collection

功能：
- 文字 / 圖片頁籤切換
- 文字卡片點擊複製
- 文字卡片長按修改 / 刪除 / 排序
- 圖片圓形縮圖
- 圖片點擊分享
- 圖片長按刪除 / 排序
- 新增文字與加入照片
