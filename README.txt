Collection PWA PhotoGrid V3

本版更新：
- 照片改成一列四張
- 照片自動往下排列
- 不再左右滑動
- 保留加入照片、分享、長按管理、備份還原、清除全部

上傳方式：
1. 解壓縮 ZIP
2. 將全部檔案覆蓋上傳到 GitHub Repository
3. 確認 index.html、manifest.json、sw.js、icon-192.png、icon-512.png 都有更新
4. GitHub Pages 開啟後，iPhone 建議刪除舊主畫面 App 再重新加入

重要：
如果 iPhone 還顯示舊版，請到：
設定 → Safari → 進階 → 網站資料 → 刪除你的 GitHub Pages 網站資料
然後重新加入主畫面。

本版畫面會顯示：版本：PhotoGrid V3
看到這行才代表新版本載入成功。
